# Scraping amazon jobs to get notification about the job availability
 ## Technologies used
    - python
    - selenium

## Steps taken for project setup
    - installing selenium 
    - pip install selenium
    - install chromedriver
    - for mac user put the chromedriver inside /usr/local/bin
